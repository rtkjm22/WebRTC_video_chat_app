## 実行方法（所要時間：約5分）

#### ①ソースコードをクローン
```terminal
git clone https://github.com/rtkjm22/WebRTC_video_chat_app.git
```
#### ②クローンしたディレクトリへ移動
```terminal
cd webrtc-nest-videochat
```

#### ③パッケージインストール
```terminal
npm install
```
#### ④プロジェクトの立ち上げ
```terminal
npm run start
```

#### ⑤ビデオチャットにアクセス
```
http://localhost:3000
```

#### ⑥別ブラウザ（同じブラウザならシークレットモードも可）でもアクセス
```
http://localhost:3000
```
